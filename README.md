# API Crawler Docs

一个轻量级的 API 爬虫接口文档管理工具。

## 功能特点

- 🚀 **一键启动** - 运行一个命令即可启动服务和打开浏览器
- 💾 **自动保存** - 数据自动保存到本地 JSON 和 Markdown 文件
- 🤖 **AI 友好的** - 生成的文档格式方便 AI 分析和使用
- 📋 **CRUD 操作** - 支持新增、编辑、删除接口
- 🔍 **搜索过滤** - 快速查找接口

## 快速开始

```bash
npx api-crawler-docs
```

## 使用方法

1. 启动后会自动打开浏览器访问 `http://localhost:3456`
2. 点击"新增接口"添加新的 API
3. 粘贴 cURL 命令和响应内容
4. 保存后自动生成文档

## 输出文件

启动后会在当前目录下创建 `doc` 文件夹：

```
doc/
├── apis.json       # JSON 格式数据
└── 爬虫文档.md     # Markdown 格式文档
```
